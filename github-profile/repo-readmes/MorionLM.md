## What is MorionLM?

Most note-taking apps store your thoughts. **MorionLM synthesizes them.**

It's a node-based second brain where every thought you capture becomes part of a living knowledge graph. The AI doesn't just retrieve — it identifies the hidden connections and overlooked patterns across everything you've recorded.

Think of it as a personal intelligence layer: your ideas go in, insight comes out.

---

## ✨ Features

### 🔵 Node-Based Thought Capture
Record discrete thoughts, ideas, or context snippets as individual nodes. Build your knowledge graph organically over time — no rigid folder structures, no taxonomies to maintain.

### ⚡ AI Pattern Synthesis
Select multiple nodes and hit **"Synthesize Selected"** — the AI identifies:
- Hidden connections between unrelated ideas
- Recurring themes and blind spots
- Patterns you're consistently missing

### 💬 Ask Your Brain
The right panel lets you query your own knowledge base with natural language:
> *"What do I know about X?"*  
> *"What are the contradictions in my thinking about Y?"*

It remembers everything you've captured — so you don't have to.

### 🎨 Distraction-Free Interface
- Minimal dark theme with glassmorphism panels
- Left panel for capture, right panel for AI insights
- Zero friction between idea and record

## 🚀 Getting Started

**Online:** Visit **[morion-lm.vercel.app](https://morion-lm.vercel.app)** — no account required.

**Local:**
```bash
git clone https://github.com/vsmrj-hash/MorionLM
cd MorionLM
npm install
npm run dev
# Open http://localhost:3000
```

---

## 🔄 How to Use

1. **Capture** — Type a thought in the left panel and hit "Record Capture"
2. **Build** — Add more nodes over time. Each one enriches the graph.
3. **Synthesize** — Select multiple nodes → "Synthesize Selected" to surface patterns
4. **Query** — Ask a question in the right panel chat interface
