## 📋 How to Apply Topics in GitHub

1. Go to the repository page
2. Click the **⚙️ gear icon** next to "About" (top right of the repo, below the "Code" button)
3. Update the **Description** and **Website**
4. In the **Topics** field, type each tag and press Enter
5. Click **Save changes**

Takes ~2 minutes per repo. 5 repos = 10 minutes total.
