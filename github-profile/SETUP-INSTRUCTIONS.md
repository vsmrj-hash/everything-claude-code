# 🚀 GitHub Profile Setup Instructions
### Everything you need to make your profile recruiter-ready

## Step 1 — Create Your Profile README (Most Impactful)

GitHub shows a special README when you create a repo with the same name as your username.

1. Go to **[github.com/new](https://github.com/new)**
2. Set **Repository name** to exactly: `vsmrj-hash`
3. Check **"Add a README file"**
4. Click **"Create repository"**
5. Click the **pencil icon** to edit `README.md`
6. **Delete all existing content**
7. **Paste the entire content** from `github-profile/README.md` in this repo
8. Click **"Commit changes"**

✅ Your profile at `github.com/vsmrj-hash` will now show a beautiful landing page.
