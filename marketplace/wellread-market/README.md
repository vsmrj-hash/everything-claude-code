# WellRead — Health & Wellness Ebook Marketplace

> Science-backed guidance for your body, your life.

A full-stack Next.js 14 marketplace for health & wellness ebooks — GLP-1 transitions, perimenopause, caregiver support, family safety, and more.

## Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 14 App Router |
| Styling | Tailwind CSS v4 |
| Auth | Clerk |
| Database | Supabase + Drizzle ORM |
| Payments | Stripe Checkout |
| Deployment | Vercel |

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/vsmrj-hash/everything-claude-code&root-directory=marketplace/wellread-market)

Or manually:
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import `vsmrj-hash/everything-claude-code`
3. Set **Root Directory** to `marketplace/wellread-market`
4. Deploy

## Environment Variables

Copy `.env.example` to `.env.local` and fill in:

```bash
cp .env.example .env.local
```

| Variable | Where to get it |
|----------|----------------|
| `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` | [clerk.com](https://clerk.com) → Create Application |
| `CLERK_SECRET_KEY` | Same |
| `NEXT_PUBLIC_SUPABASE_URL` | [supabase.com](https://supabase.com) → Project → Settings → API |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Same |
| `DATABASE_URL` | Supabase → Settings → Database → Connection String |
| `STRIPE_SECRET_KEY` | [stripe.com](https://stripe.com) → Developers → API keys |
| `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` | Same |
| `STRIPE_WEBHOOK_SECRET` | Stripe → Developers → Webhooks → Add endpoint |

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

The app runs fully on static seed data without any API keys. Add keys progressively to activate each service.
